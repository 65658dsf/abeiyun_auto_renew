# 阿贝云自动续费脚本

这是一个用于自动登录阿贝云账号并进行免费服务器延期操作的Python脚本。

## 功能特性

- 🔐 自动登录多个阿贝云账号
- 🔄 自动进行免费服务器延期操作
- 📝 详细的日志记录
- ⚡ 支持批量处理多个账号
- 🛡️ 错误处理和异常恢复
- 📋 自动填写延期表单（发帖地址、上传截图、提交）
- 🔄 多账号切换和自动退出登录
- 🔁 智能重试机制（每个账号最多3次尝试）
- ⏳ 表单加载等待机制，确保完全加载后再操作
- 📊 智能状态识别（支持"延期申请等待审核中"等状态）

## 环境要求

- Python 3.7+
- Microsoft Edge 浏览器
- Windows 操作系统

## 安装步骤

1. **安装Python依赖包**
   ```bash
   pip install -r requirements.txt
   ```

2. **确保已安装Microsoft Edge浏览器**
   - 脚本会自动下载和配置EdgeDriver

## 使用方法

1. **配置用户信息**
   - 编辑 `users.json` 文件，添加你的阿贝云账号信息
   - 格式如下：
   ```json
   {
       "手机号1": {
           "password": "密码1",
           "lastsign": "2020-02-20 14:24:13"
       },
       "手机号2": {
           "password": "密码2",
           "lastsign": "2020-02-20 14:24:13"
       }
   }
   ```

2. **运行脚本**
   ```bash
   python abeiyun_auto_renew.py
   ```

## 脚本工作流程

1. **读取用户配置** - 从 `users.json` 文件加载账号信息
2. **启动Edge浏览器** - 自动配置并启动浏览器
3. **循环处理每个账号**（每个账号最多重试3次）：
   - 打开阿贝云登录页面 (`https://www.abeiyun.com/login/`)
   - 自动填写用户名和密码
   - 点击登录按钮
   - 导航到免费服务器列表页面 (`https://www.abeiyun.com/control/#/freeServerList`)
   - 查找并点击"免费延期"按钮
   - 智能等待表单完全加载（最多15秒）
   - 填写延期表单：
     - 输入发帖地址：https://blog.csdn.net/qwdasfweq/article/details/142734456?spm=1001.2014.3001.5501
     - 上传截图文件：d:/柠檬的发明/阿贝云自动续费/ag.png
     - 点击提交按钮
   - 智能识别提交状态：
     - ✅ "延期申请等待审核中" - 提交成功，等待审核
     - ✅ "提交成功" - 立即成功
     - ❌ 其他错误状态 - 触发重试机制
   - 安全退出当前账号登录
   - 更新最后签到时间
4. **生成日志** - 所有操作都会记录到 `abeiyun_auto_renew.log` 文件

## 日志文件

脚本运行时会生成详细的日志文件 `abeiyun_auto_renew.log`，包含：
- 登录状态
- 延期操作结果
- 错误信息
- 时间戳

## 故障排除

### 常见问题

1. **登录失败**
   - 检查用户名和密码是否正确
   - 确认账号没有被锁定
   - 检查网络连接

2. **找不到延期按钮**
   - 可能服务器已经延期过了
   - 页面结构可能发生变化
   - 检查是否有免费服务器

3. **浏览器启动失败**
   - 确保已安装Microsoft Edge浏览器
   - 检查网络连接（需要下载EdgeDriver）
   - **如果自动下载失败**：
     - 请手动下载对应版本的 [Microsoft Edge WebDriver](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)
     - 解压并将 `msedgedriver.exe` 放到脚本同级目录下
     - 再次运行脚本即可
